The BYU Ruby Users Group was kind enough to allow me to give my Copland presentation this evening, as practice for the up-coming Ruby Conference.

There were only four attendees, besides myself, which is about par for the course when it comes to attendance at BYU RUG meetings. Still, I was flattered to learn that one attendee, Jacob Fugal, is actually using Copland, and using it to write a Ruby version of my Dungeon Generator, no less!

The feedback I got was wonderful. They all felt the presentation was valuable, and had good comments to make on it. I’ll be incorporating their suggestions tonight. I actually feel like I’ll be ready to give this presentation next week!
