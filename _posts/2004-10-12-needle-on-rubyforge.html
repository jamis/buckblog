Needle is now on RubyForge!

That is to say, the source code is in CVS on RubyForge… still no packaged releases, but now that it is in CVS I feel better about hacking on it. I can now rollback to previous versions if my experiments go sour. :)

(For those of you that don’t know: Needle is my new dependency injection container for Ruby. It’s coming along quite nicely!)
